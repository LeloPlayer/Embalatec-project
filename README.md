# 📦 Site Institucional – Embalatec (Projeto Front-End)

## 📌 Sobre o Projeto
Este projeto consiste no desenvolvimento de um site institucional interativo inspirado na empresa onde atuo atualmente.

A aplicação foi construída com base em processos reais da empresa, como produção, expedição e qualidade, com o objetivo de transformar informações operacionais em uma interface digital clara, moderna e funcional.


## 🚀 Tecnologias Utilizadas
- HTML5
- CSS3
- JavaScript


## 🎯 Objetivos do Projeto
- Aplicar conhecimentos de desenvolvimento front-end em um contexto real
- Criar uma interface moderna e responsiva
- Transformar processos empresariais em uma experiência digital intuitiva
- Fortalecer portfólio profissional com projeto prático


## 💡 Funcionalidades
- Layout responsivo (Mobile First)
- Navegação intuitiva
- Estrutura de páginas institucionais
- Seções baseadas em áreas reais da empresa:
  - Produção
  - Expedição
  - Qualidade
  - Processos internos
- Interações com JavaScript


## 🧠 Aprendizados
Durante o desenvolvimento, foram aplicados e reforçados:

- Estruturação semântica com HTML
- Estilização moderna com CSS (Flexbox/Grid)
- Responsividade
- Manipulação do DOM com JavaScript
- Organização e boas práticas de código


## 🛠️ Melhorias Futuras
- Integração com back-end (Node.js ou API REST)
- Formulário de contato funcional
- Implementação de SEO
- Acessibilidade (A11Y)
- Otimização de performance
